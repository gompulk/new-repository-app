# new-repository-app



Aplikasi Node.js sederhana (HTTP API) siap rilis.



## Fitur

- Endpoint root `GET /`
- 
- Endpoint health check `GET /health`
- 
- Konfigurasi port lewat environment variable `PORT`
- 


## Menjalankan lokal

1. Install Node.js 18+
2. 
2. Jalankan `npm install`
3. 
3. Jalankan `npm start`
4. 


Aplikasi berjalan di `http://localhost:3000` (default).



## Script

- `npm start`
- 
- `npm run dev`
- 
- `npm test`
